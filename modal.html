The Modal Window

We use modals a lot at NGI to show product copy, legal content, and even in Cortana to get quotes. What I need is the javascript end figured out. To the right is a set of requirements for this project which was heavily influenced by the way modals work in the Foundation Framework.

  • Modals need to be able to be launched from a variety of different places, including anchors, divs and spans.
  • There will also likely be many modals on a page, so the use of a class identifier or data attribute is going to be key
  • The modals will need to be smart enough to close the first one if a second one is launched from the first modal
  • The close function is going to need to be variable, sometimes modals can be closed by clicking the background, sometimes it wont be able to
